    • Selectors Content Filters
      • :contains()

        • :contains() Selector

        The :contains() selector selects elements containing the specified string.

        jQuery 
         
Select all < h1 > elements containing "WalkMe":

wmjQuery('h1:contains(WalkMe)')
      • :empty

        • :empty Selector

        The :empty selector selects empty elements. An empty element is an element without child elements or text.

        jQuery 
         
wmjQuery(':empty')
      • :has()

        • :has() Selector

        The :has() selector selects all elements that have one or more elements inside of them, that matches the specified selector.

        jQuery 
         
wmjQuery("p:has(span)")
      • :parent

        • :parent Selector

        The :parent selector selects all elements that are the parent of another element, including text nodes.

        jQuery 
         
Select all < td > elements with children, including text:

wmjQuery("td:parent")
    • Extensions DeepUI
      • :deepui_visible

        • :deepui_visible

        Checks that the element has height and width and is on the top/obscured (not covered in z-index). Use in cases where mt_visible is not strong enough.

        jQuery 
         
wmjQuery('td:deepui_visible')
      • :deepui_hidden

        • :deepui_hidden

        Used in the opposite way of deepui_visible

        jQuery 
         
wmjQuery('td:deepui_hidden')
      • :textStartsWith()

        • :textStartsWith()

        Checks that text of element starts with input. Note that text is defined as wmjQuery(elem).text()

        jQuery 
         
wmjQuery('td:textStartsWith(Account)')
      • :textEndsWith()

        • :textEndsWith()

        Checks that text of element ends with input.

        jQuery 
         
wmjQuery('td:textEndsWith(ing)')
      • :hasParent()

        • :hasParent()

        Verify that input element is an ancestor (parent, grandparent etc) of element

        jQuery 
         
 wmjQuery(':hasParent(.labelCol)')
      • :directContains()

        • :directContains()

        Similar to :contains() but checks only direct children and not all descendants

        jQuery 
         
wmjQuery('.labelcol:directContains(Account)')

    • Attributes Attributes
      • [name|=”value”]

        • Attribute Contains Prefix Selector [name|=”value”]

        Selects elements that have the specified attribute with a value either equal to a given string or starting with that string followed by a hyphen (-).

        jQuery 
         
Select all < p > elements with a title attribute that 
starts with the value "Tomorrow":

wmjQuery("p[title|='Tomorrow']")
      • [name*=”value”]

        • Attribute Contains Selector [name*=”value”]

        Selects elements that have the specified attribute with a value containing a given substring.

        jQuery 
         
Select all < input > elements with a name attribute 
that contains the word "nation":

wmjQuery("input[name*='nation']")
      • [name~=”value”]

        • Attribute Contains Word Selector [name~=”value”]

        Selects elements that have the specified attribute with a value containing a given word, delimited by spaces.

        jQuery 
         
Select all < input > elements with a name attribute 
that contains the specific word "nation":

wmjQuery("input[name~='nation']")
      • [name$=”value”]

        • Attribute Ends With Selector [name$=”value”]

        Selects elements that have the specified attribute with a value ending exactly with a given string. The comparison is case sensitive.

        jQuery 
         
Select all < a > elements with a href attribute that 
ends with ".org":

wmjQuery("a[href$='.org']")
      • [name=”value”]

        • Attribute Equals Selector [name=”value”]

        Selects elements that have the specified attribute with a value exactly equal to a certain value.

        jQuery 
         
Select every element containing an input attribute with 
the value "price":

wmjQuery("[id='price']")
      • [name!=”value”]

        • Attribute Not Equal Selector [name!=”value”]

        Select elements that either don't have the specified attribute, or do have the specified attribute but not with a certain value.

        jQuery 
         
Select all < p > elements NOT containing a class 
attribute with the value "intro":

wmjQuery("p[class!='intro']")
      • [name^=”value”]

        • Attribute Starts With Selector [name^=”value”]

        Selects elements that have the specified attribute with a value beginning exactly with a given string.

        jQuery 
         
Select all < input > elements with a name attribute 
that starts with "nation":

wmjQuery("input[name^='nation']")

    • Traversing Filtering
      • .filter()

        • .filter() Method

        The filter() method lets you specify a criteria. Elements that do not match the criteria are removed from the selection, and those that match will be returned.

        jQuery 
         
Return all < p > elements with class name "intro":

wmjQuery("p").filter(".intro")
    • Traversing Traverse
      • .find()

        • .find() Method

        Finds all the descendant elements of the selected element. It will traverse all the way down to the last leaf of the selected element in the DOM tree.

        jQuery 
         
Select all the < span > elements connected to the .button class

wmjQuery(".button").find("span")
      • .children()

        • .children() Method

        The children() method returns all direct children of the selected element.

        jQuery 
         
Return elements that are direct children of < ul >:

wmjQuery("ul").children()
      • .closest()

        • .closest() Method

        For each element in the set, get the first element that matches the selector by testing the element itself and traversing up through its ancestors in the DOM tree.

        jQuery 
         
Return the first ancestor of < span >, that is an < ul > element:

wmjQuery("span").closest("ul")
      • .next()

        • .next() Method

        Get the immediately following sibling of each element in the set of matched elements. If a selector is provided, it retrieves the next sibling only if it matches that selector.

        jQuery 
         
Return the next sibling element of each < li > element with class name "start":

wmjQuery("li.start").next()
      • .prev()

        • .prev() Method

        Get the immediately preceding sibling of each element in the set of matched elements. If a selector is provided, it retrieves the previous sibling only if it matches that selector.

        jQuery 
         
Return the previous sibling element of each < li > element with class name "start":

wmjQuery("li.start").prev()
